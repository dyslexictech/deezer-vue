# Deezer Vue
> Deezer client built with Vue\Vuex [https://developers.deezer.com/api](https://developers.deezer.com/api) 

Live [https://sh0cked.github.io/deezer-vue/](https://sh0cked.github.io/deezer-vue/)

## Installation

1. Clone repo
2. run ```yarn```
3. run ```yarn run serve```
